# 🔔 VSC에서 md(markdown)파일 다루기

1. 목차 만들기

   ```
   ctrl + shift + ]
   ```

---

2. 글씨 Bold / Italic

   **_md파일_**

   ```
   ctrl + B / I
   ```

---

3. 글씨 취소선

   ~~md파일~~

   ( 빈 라인에 하면 라인 전체가 검은색이 됨 👇🏻)

   ```
   alt + S
   ```

---

4. md파일 수학환경 전환 ( 1+1=2 )

   $1+1=2$

   ```
   ctrl + M
   ```

---

5. 작업 목록 항목 선택/취소

   - [ ] md파일
   - [x] md파일 ( alt + C 누르면 선택해제)
   - [ ] md파일 ( alt + C 누르면 선택)

   ```
   alt + C
   ```

---

6. 미리보기 전환

   ```
   ctrl + shift + V
   ```

---
