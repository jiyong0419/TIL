# 🔔 모던 자바스크립트 Deep Dive 6장~7장

**6장 데이터 타입**

number / string / boolean / undefined / null / object / function / array

```
6-1. number타입
    - 모든 수를 실수로 처리한다. ( 1 === 1.0 )
    - Infinity , -Infinity , NaN도 number타입이다.

6-2. string타입
    - 작은 따옴표, 큰 따옴표, 백틱기호를 사용한다.

6-3. 템플릿 리터럴
    - 백틱기호를 사용한 문자열을 의미한다.
    - 백틱기호 안에 줄바꿈과 들여쓰기는 적용이 된다.
    - 표현식을 사용하려면 ${ }로 감싸준다
    - 이스케이프 시퀀스 https://ponyozzang.tistory.com/52

6-4. boolean타입
    - true 와 false

6-5. undefined타입
    - 변수를 선언하고 값을 할당하지 않으면 undefined
    - 개발자의 의도된 값이 아닌 자바스크립트 엔진에 의해 초기화 된 값이다.

6-6. null타입
    - 변수에 값이 없음을 의도적으로 명시한 값
    - 함수가 유효한 값을 return할수 없는 경우 명시적으로 null을 반환한다.

6-8. object타입
    - { key : value, ... }

6-9. 데이터 타입의 필요성
    - 값을 저장할 때 메모리 공간의 크기를 결정함.
    - 값을 참조할 때 읽어들일 메모리 공간의 크기를 결정함.
    - 읽어들인 2진수를 어떻게 해석할지 결정함.
        < 0100 0001은 number타입으로는 65, string타입으로는 'A'가 된다. >

6-10. 동적 타입 언어와 정적 타입 언어
    - C 나 자바는 타입을 선언해야하는 정적 타입언어다. 한번 선언된 타입은 변경 불가!
    - 자바스크립트는 타입을 선언하지 않고, 할당에 의해 타입이 변경되는 동적 타입 언어!
```

**7장 연산자**

연산의 대상을 피연산자라고 함.
피연산자는 표현식이어야 함.

```
7-1. 산술연산자
    - 산술연산자는 수학적 계산을 수행함.
    - 산술연산이 불가능하면 'NaN'을 반환함.
    - 피연산자의 개수에 따라 이항산술연산자 or 단항산술연산자로 구분
    - 이항산술연산자 +, -, *, /, %
    - 단항산술연산자 ++, --, +, - ( +,-는 양,음수기호 )
    - 증감연산자( ++,-- )는 피연산자의 값을 변경한다.
        < 위치에 따라 전위/후위 증감연산자로 구분 >
    - 양/음수기호( +,- )는 number타입이 아닌 피연산자도 number타입으로 변환하여 반환함.
        < 피연산자 값이 바뀌는건 아니고 반환값만 바뀐다 >
    - +연산자는 피연산자 중 string 타입이 있다면 문자열 연결 연산자로써 string을 반환함.

7-2. 할당연산자
    - 할당연산자 =, +=, -=, *=, /=, %=
    - 할당문은 표현식이다.
    - x += 5 는 x = x+5 랑 같다.

7-3. 비교연산자
    - 비교연산자는 boolean값을 리턴한다.
    - 비교연산자 ===, !==, >, >=, <, <=
    - NaN은 비교연산자로 비교가 불가능하다.
        < isNaN 함수를 사용한다 >
        < Object.is(NaN,NaN) = true // Object.is(+0,-0) = false >

7-4. 삼항조건연산자
    - 조건식 ? ture : false
    - 삼항조건연산자는 표현식이다.

7-5. 논리연산자
    - ||, &&, !
    - 피연산자가 boolean이 아니면 boolean으로 암묵적 타입 변환을 하여 연산한다.
    - 논리연산자의 반환값은 boolean이 아닌 경우도 있다.
        < 'Cat' && 'Dog' = 'Dog' // 'Cat' || 'Dog' = 'Cat' >

7-6. 쉼표연산자
    - (,)

7-7. 그룹연산자
    - 소괄호, 우선순위 1등이다.

7-8. typeof연산자
    - 피연산자의 타입을 string으로 반환한다.
    - null타입은 object로 반환된다.
    - 선언한적 없는 식별자는 undefined로 반환된다.

7-9. 지수연산자 (추후 설명)
    - 지수연산자 **
    - 좌항이 밑, 우항이 지수 ( 2**20 = 2의 20승 )
    - 음수를 밑으로 사용하려면 밑을 소괄호로 감싸준다.

7-10. 그 외 연산자 ( 추후 설명 )
    - ?., ??, delete, new, instanceof,in

7-12. 연산자 우선순위
    - 그룹연산자를 사용하여 우선순위를 조절하자.
```
