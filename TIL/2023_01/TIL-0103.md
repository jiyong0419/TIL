# 🔔 모던 자바스크립트 Deep Dive 11장

**11장 원시값과 객체의 비교**

```
11-1. 원시값
    - 변경 불가능한 값
      <변수가 참조하는 메모리 주소의 메모리 셀안에 값을 바꿀수 없다는 뜻>
    - 숫자값은 몇이 되었든 무조건 8바이트 크기
    - 문자열은 1개의 문자당 2바이트 크기
    - 문자열은 유사배열객체로써 인덱스를 통해 각 문자에 접근 할 수 있고,
      length 프로펕티를 갖으며, for문을 이용하여 순회할 수 있다.
      <하지만, 어떤 방법으로든 문자열의 구성문자를 변경할 수 없다.>
    - 변수 x에 원시값 1을 갖는 변수 y를 할당하면 y의 원시 값이 다른 메모리(x)에 전달되고,
      x와 y는 서로 상호작용하지는 않는다.

11-2. 객체
    - 객체는 프로퍼티 개수에 제약도 없고, 프로퍼티 값의 제약도 없다.
      언제든 프로퍼티를 동적으로 추가/삭제 할수 있으며 객체마다 크기는 제각각이다.
    - 객체는 변경 가능한 값
    - 변수 obj는 0x000001 메모리 주소를 참조하고
      그 안에는 0x000010 메모리주소가 들어있고 해당 메모리 주소는 객체가 담겨있다.
      이때 obj2라는 변수에 obj를 할당하면 obj2는 0x000002라는 메모리 주소를 참조하고
      그 안에 0x000010이라는 메모리 주소가 담긴다.
      obj와 obj2는 동일한 객체메모리주소를 참조하는 각기다른 메모리를 참조하며, 서로 상호작용이 된다.

```
