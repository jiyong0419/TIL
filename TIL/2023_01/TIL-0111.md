# 🔔 모던 자바스크립트 Deep Dive 19장(2)

**19장 프로토타입**

```
19-4. 리터럴 표기법에 의해 생성된 객체의 생성자 함수와 프로토타입.
    - 리터럴 표기법에 의해 생성된 객체도 프로토 타입을 갖는다.
      prototype내부에는 constructor(생성자) 프로퍼티가 있어야하고,
      리터럴 표기법에 의해 생긴 객체도 prototype의 constructor 프로퍼티에
      가상의 생성자 함수가 할당된다.
      즉, prototype과 생성자 함수는 언제나 쌍으로 존재한다.

19-5. 프로토타입의 생성시점
    - 프로토타입객체는 생성자 함수가 생성되는 시점에 더불어 생성됨 (둘은 한쌍이니까)
      생성자함수는 사용자 정의 생성자 함수(new Person)과 빌트인 생성자 함수(new Object,...)로
      구분할 수 있다.
    - 사용자 정의 생성자 함수는 함수 선언문으로 정의 되었다면 런타임 이전 시점에,
      함수 표현식으로 정의 되었다면 함수 표현식이 런타임 되는 시점에 프로토타입 객체가 생성된다.
    - 생성자 함수의 프로토타입 객체도 프로토타입 객체를 갖고있으며 그 값은 Object.prototype이다.
    - 빌트인 생성자 함수는 전역객체(window,global)가 생성되는 시점에 생성되고,
      더불어 빌트인 생성자 함수의 프로토타입 객체도 같이 생성된다.
    - 빌트인 생성자 함수로 객체가 생성되기 이전에 이미 빌트인 생성자 함수랑 프로토타입 객체는 존재한다.
      이후 빌트인 생성자 함수 혹은 리터럴 표기법으로 객체를 생성하면
      이미 존재하는 빌트인 함수의 프로토타입이 생성된 객체의 [[Prototype]] 내부슬롯
      즉, 생성된 객체의 프로토타입 객체에 할당이 된다.
19-6. 객체 생성방식과 프로토타입의 결정
    - 객체 생성방식
      > 객체 리터럴 ( {} ),
        Object 생성자함수 ( new Object() ),
        생성자 함수 ( new Person() ),
        Object.create메서드
    - 객체 리터럴에 의해 생성된 객체의 프로토타입은 Object.prototype이다.
      객체 리터럴에 의해 생성된 객체는 Object.prototype를 상속받고,
      Object.prototype의 프로퍼티/메서드를 사용할 수 있다.
    - Object 생성자함수에 으해 생성된 객체의 프로토타입객체는 Object.prototype이고,
      객체리터럴에 의해 생성된 객체에 대한 설명과 같다.
      객체리터럴에 의해 생성된 객체와 Object 생성자함수에 의해 생성된 객체는
      프로퍼티를 추가하는 방식에서 차이점이 있을뿐이다.
    - 생성자함수에 의해 생성된 객체(인스턴스)의 프로토타입객체는
      인스턴스의 prototype 프로퍼티에 할당 되어 있는 객체다.
      (자바스크립트엔진은 인스턴스의 prototype에 암묵적으로 생성자함수의
       prototype을 할당해준다. 그래서 인스턴스.__proto__ === 생성자함수.prototype이 true인 것이다.)

19-7. 프로토타입 체인
    - 프로토타입 객체의 프로토타입 객체는 언제나 Object.prototype이다.
    - 자바스크립트엔진은 객체의 프로퍼티/메서드에 접근 할때 해당 객체에 해당 프로퍼티/메서드가 없다면
      해당 객체의 프로토타입객체에 접근하여 해당 프로퍼티/메서드를 찾고
      해당 객체의 프로토타입객체에도 없다면
      해당객체의 프로토타입객체의 프로토타입객체..마지막엔 Object.prototype까지 파도타기해서
      해당 프로퍼티/메서드를 찾아낸다. 이를 프로토타입 체인이라고 한다.
      (프로토타입 체인의 최상위에는 Object.prototype이 있다. 체인의종점)
      (Object.prototype의 prototype은 null이다.)
    - 프로토타입체인 = 상속과 프로퍼티 검색을 위한 매커니즘
      스코프체인 = 식별자 검색을 위한 매커니즘

```
